# InterviewIQ.AI

An AI-powered interview preparation platform that helps users prepare for technical interviews through resume-based question generation, mock interviews, and AI-generated feedback.

## Live Demo

https://interviewiq-vaph.onrender.com

## Features

* Secure user authentication
* Resume upload and analysis
* AI-generated interview questions
* Interactive mock interview experience
* AI-powered feedback on responses
* User-friendly and responsive interface

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Configure environment variables

Create a `.env` file and add the required API keys and configuration values.

4. Start the application

```bash
npm run dev
```
